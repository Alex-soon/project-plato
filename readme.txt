Установка и настройка.

1. Установите node.js для вашей операционной системы. Исходники можно найти на странице https://nodejs.org/en/download/
2. Зайдите в корневую директорию проекта и установите npm пакеты зависимостей из файла package.json.
   Рекомендую использовать команду npm install в консоле.
3. Установите последнюю версию браузера Google Chrome.
4. Скачайте WebDriver для браузера Google Chrome и поместите в директорию /src. Необходимую версию можно найти на странице http://chromedriver.storage.googleapis.com/index.html
   Для версии браузера Google Chrome 63.0.3239.132 и ос Windows подходит версия 2.34.
